Learning Management Systems(NIK)

"One point for all the training of Cognizant Academy."

PROJECT GOALS Implementing a digital solution for learning management of trainees

The main goal of this project was to implement a digital solution for learning management of trainees. For me, it was to learn how to work on existing code and improve the functionalities with the help of designing and programming and also to learn the technology stack that was being used for this project.

TECHNOLOGY STACK Tools to bring the project to life

The various technologies being used for this project were: Front-End we used HTML5, CSS3, Bootstrap, AngluraJS, JQuery, while on the Back-End we used C#, SQL Server, Web API were being used. For Version management and collaboration Team foundation server was being used

TEAMS The Workforce

There were four teams to begin with: Learner, SolutionDesigner, Supervisor and Quiz. Each team had access to their branch of code and after every iteration, the teams came together to merge all of their code into one. I was part of the Solution Designer and worked in collaboration with mine the other teams and inside my team. The primary objective of my team was to create interfaces where an administrator can view all the published/rejected/draft courses/LearningPath as per user requirement
